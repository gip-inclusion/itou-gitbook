# Renseigner le réseau auquel vous adhérez

Rendez-vous sur _Mon Espace_ &gt; _Mes Structures_ &gt; _Réseau_. 

Ensuite, cliquez sur le ou les réseaux auxquels vous êtes adhérent. Puis cliquez sur "Enregistrer"
